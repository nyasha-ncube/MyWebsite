# MyWebsite
This project is a modern, responsive flower-shop website designed to showcase floral products such as bouquets, gift hampers, wedding arrangements, and seasonal collections. It includes an elegant homepage, category pages, product listings, an add-to-cart animation, and a clean checkout flow.
